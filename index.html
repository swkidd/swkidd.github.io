Micro Skills

A framework for interactive learning

There are steps involved in learning any new skill. This framework aims to facilitate reasoning about skills so that they can be better broken down into parts, and targeted with interactive learning experiences.

A skill is something done. Learning a skill involvings learning how to do things, and how not to do them. A skill is a performance, a goal oriented movement which can be judged, even measured, towards how well it achieves its goal. Learning a skill, perfecting a skill, is then simply learning a sequence of movements, and under which circumstances to perform then.

Here we see a natural point at which skills can be broken down into parts. At which points in performing a skill do we have to make decisions about what sequence of movements to perform next.

To view the same question in the negative, what kind of skills do not require attention in such a way to choose between actions. Imagine a skill which can be performed without fail so long as the movements which constitute it are performed correctly. This is a skill that does not require choice. Or at least does not require choice while it is being performed, i.e. once its performance has been deemed necessary. By definition then, such a skill can be performed unconsciously. As there is, by definition, no point in the skills execution where, being aware of something, a decision would be made to act differently.

A hypothesis of this framework is then that skills are constituted of movements which can be performed unconsciously, so called perfectable skills or micro-skills, and the decisions which select between them.

Teaching is then confined to either teaching an action in such a way that it can be performed unconsciously, or teaching sensitivity towards recognizing when one action should be performed rather than another.
