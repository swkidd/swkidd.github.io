Micro Skills

A framework for interactive learning

There are steps involved in learning any new skill. This framework aims to facilitate reasoning about skills so that they can be better broken down into parts, and targeted with interactive learning experiences.

A skill is something done. Learning a skill involvings learning how to do things, and how not to do them. A skill is a performance, a goal oriented movement which can be judged, even measured, towards how well it achieves its goal. Learning a skill, perfecting a skill, is then simply learning a sequence of movements, and under which circumstances to perform then.

Here we see a natural point at which skills can be broken down into parts. At which points in performing a skill do we have to make decisions about what sequence of movements to perform next.

To view the same question in the negative, what kind of skills do not require attention in such a way to choose between actions. Imagine a skill which can be performed without fail so long as the movements which constitute it are performed correctly. This is a skill that does not require choice. Or at least does not require choice while it is being performed, i.e. once its performance has been deemed necessary. By definition then, such a skill can be performed unconsciously. As there is, by definition, no point in the skills execution where, being aware of something, a decision would be made to act differently.

A hypothesis of this framework is then that skills are constituted of movements which can be performed unconsciously, so called perfectable skills or micro-skills, and the decisions which select between them.

Teaching is then confined to either teaching a micro-skill in such a way that it can be performed unconsciously, or teaching sensitivity towards recognizing when one action should be performed rather than another.

Movement

Skills consist of micro-skills, and decisions that select between them. These micro-skills, being able to be preformed unconcioulsy, consist solely of movements. For sake of reasoning, we break movement into 3 more or less non-overlapping categories. Eye movement, speech and bodily movement. Sequences of actions comprising micro-skills take place in these 3 tracks. To best teach a skill interactively it is best to target each category seperately. Regard physical action and non-physical action in each category as the same. If a micro-skill requires sub-vocal speech, imagined bodily movement etc. to be preformed regard that as being the same type as actual action.

Learning as a skill

The mediumn through which a skill is taught, and the process of learning a skill, will enevitably involve learning skills which are not needed when actually preforming the skill. Caution must be taken not add points of decision where a choice is between an action which prefects usage of the medium of learning and the skill in question. For instance, a mulitple choice question where context in the question, in place of knowledge of the actual skill, allows choices to be eliminated.

Decisions and Labeling

The choice between mico-skills is mastered in so far as it is a function. The inputs to this function are the infomation which, having been recognized, a choice can be made. Being a funciton then, such a choice defines an equivelence among this information and labels it. In this way, the information that decides a decision stands for, becomes synonomous with, the micro-skill that flows from it.
